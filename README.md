# Personal Expense Tracker CLI

## Overview

Personal Expense Tracker is a command-line Python application that helps users manage their daily expenses. It allows users to add, view, search, edit, delete, and summarize expenses. All data is stored locally in a CSV file.

## Features

- Add new expense
- View all expenses
- Search expenses
- Calculate total expense
- Category-wise summary
- Monthly summary
- Edit existing expense
- Delete expense
- CSV file storage
- Input validation

## Technologies Used

- Python 3
- CSV Module
- OS Module
- Datetime Module

## Project Structure

```
Expense_Tracker/

```

## How to Run

1. Install Python 3.
2. Open the project folder.
3. Run:

```bash
python main.py
```
